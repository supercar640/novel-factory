# 플롯 아웃라인

## 전체 시놉시스

강남의 IT 개발자 한재현은 어느 날 밤 퇴근길에 원인 불명의 현상에 휘말려 사이버펑크 세계 '메가시티 서울'의 최하층 하부도시에 떨어진다. 신원도 크레딧도 없이 버려진 그는 정보 브로커 칼의 도움으로 간신히 생존하고, 전설적 시술사 파라셀수스를 만나 자신의 임플란트에 숨겨진 잠재력을 각성시킨다. 디버거 '고스트'로 이름을 알려가며 글리치의 단서를 추적하는 재현은, 메가코프 아크(ARC)가 글리치를 비밀리에 연구하고 있다는 사실에 다가간다. 귀환의 가능성이 열리는 동시에, 하부도시에서 쌓아온 유대와 자신의 정체성 사이에서 최종 선택을 해야 한다.

---

## 4부 구성

### 1부: 낙하 (1~5화)

**핵심**: 전이, 생존, 첫 발판

| 화 | 내용 |
|----|------|
| 1화 | 강남 퇴근길 일상 → 원인 불명의 공간 왜곡 → 하부도시 낙하. 혼란과 공포 |
| 2화 | 하부도시의 적대적 환경. 소매치기, 위협. 칼에게 발견됨. 이용가치를 본 칼이 임시 은신처 제공 |
| 3화 | 하부도시의 규칙을 배움. 더스트(암호화폐), 영역, 금기. 임플란트의 미약한 AR 노이즈를 처음 인지 |
| 4화 | 칼의 심부름으로 녹사평 시장 첫 방문. 위기 상황에서 임플란트가 순간적으로 강하게 반응 — 시스템의 '결함'이 보임 |
| 5화 | 칼이 파라셀수스에게 데려감. 파라셀수스가 임플란트를 분석하고 경악. "이건... 본 적 없는 구조야." 1차 튜닝 시작 |

**1부 종결 시점**: 주인공이 하부도시에 최소한의 발판을 마련하고, 임플란트의 잠재력을 인지

---

### 2부: 각성 (6~15화)

**핵심**: 디버거 성장, 하부도시 세력과의 관계, 글리치 단서 수집

| 구간 | 내용 |
|------|------|
| 6~7화 | 파라셀수스의 튜닝 완료. 임플란트 1차 각성 — 디지털 시스템의 버그를 시각적으로 인지하는 능력 발현. 디버거 견습 시작 |
| 8~9화 | 첫 디버거 의뢰 수행. 을지 골목의 해커들에게 '고스트'라는 이름이 붙음. 칼과의 파트너십 공고화 |
| 10~11화 | 블랙아웃과의 첫 마찰. 노의 영역에서 무허가 디버깅 → 경고. 노가 고스트에게 관심을 보임 |
| 12~13화 | 글리치에 대한 도시전설 수집. 하부도시 노인에게서 "예전에 글리치를 본 적 있다"는 증언 확보. 특정 구역에서 글리치 흔적 발견 |
| 14~15화 | 칼의 과거가 드러남 — 시너지 내부 비리 목격 후 추방. 칼이 재현을 신뢰하기 시작. 동시에 노가 '거래'를 제안: 블랙아웃을 위해 일하면 보호해주겠다 |

**2부 종결 시점**: 디버거 고스트로 하부도시에 이름을 알림. 글리치의 실체에 대한 첫 실마리 확보. 블랙아웃과의 관계가 긴장 속 공존으로 전환

---

### 3부: 추적 (16~25화)

**핵심**: 글리치 추적, 상부도시 침투, 아크(ARC)의 비밀

| 구간 | 내용 |
|------|------|
| 16~17화 | 글리치 흔적을 추적하다 아크(ARC)의 그림자를 포착. 경계층에서 아크의 비밀 데이터 패킷 발견 |
| 18~19화 | 유리와의 첫 접촉. 유리가 하부도시에 비밀리에 내려와 데이터를 수집하던 중 재현과 조우. 재현이 '다른 세계에서 왔다'는 사실에 유리가 경악 |
| 20~21화 | 유리를 통해 '프로젝트 리프트'의 존재를 알게 됨. 상부도시 침투 계획 수립. 칼의 옛 인맥과 파라셀수스의 기술 지원 |
| 22~23화 | 상부도시 첫 침투. 아크 외곽 시설에서 글리치 관련 데이터 확보. 동시에 재현의 존재가 아크 보안팀에 포착 |
| 24~25화 | 임플란트 2차 각성 — 글리치 에너지에 노출되며 능력 도약. 동시에 충격적 발견: 재현의 전이 자체가 아크의 프로젝트 리프트 실험에 의한 것일 수 있다 |

**3부 종결 시점**: 글리치의 실체와 아크의 비밀에 접근. 귀환의 가능성이 구체화되지만, 대가가 무엇인지 아직 불명

---

### 4부: 결전 (26~35화)

**핵심**: 최종 선택, 대결, 귀결

| 구간 | 내용 |
|------|------|
| 26~27화 | 아크의 프로젝트 리프트 전모 파악. 글리치를 인위적으로 생성하려는 실험이 하부도시에 재앙을 초래할 수 있다는 사실 |
| 28~29화 | 노가 아크와의 암거래를 통해 세력 확장을 꾀함. 하부도시 내 세력 재편. 재현은 칼, 파라셀수스와 함께 대응 |
| 30~31화 | 유리가 아크를 배신하고 재현 편으로. 프로젝트 리프트의 핵심 데이터를 가져옴. 귀환의 열쇠와 그 대가가 드러남 |
| 32~33화 | 아크의 최종 실험 저지를 위한 작전. 상부도시와 하부도시를 넘나드는 최종 대결 |
| 34~35화 | 재현의 최종 선택. 귀환할 것인가, 이 세계에 남을 것인가. 에필로그 |

**4부 종결 시점**: 스토리 완결

---

## 진행 상황
- **현재**: 1부 진행 중 (1~3화 초안 완료, PD 검토 대기)
- **완성된 에피소드 (초안)**:
  - 1화 「노이즈」: 강남역 퇴근길 → 시야 노이즈 → 골목이 변형 → 하부도시 녹사평 시장 낙하 → 블랙아웃 추격 → 최하층에 은신
  - 2화 「공짜는 없다」: 칼에게 발견됨 → 하부도시 투어 → 더스트 계좌 생성 → 파라셀수스 클리닉 앞에서 노이즈 감지 → 칼과 파트너십 형성
  - 3화 「결함」: 녹사평 시장 전력 오작동 → 임플란트 능력 첫 발현(시스템 결함 시각화) → 접지 라인 수동 복구 → 파라셀수스 조우 → 임플란트 분석 → 1차 튜닝 시작
